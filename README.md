## 📖 Project Summary

**Spring Boot FAQ Chatbot** is a simple question‑answering application built with **Spring Boot, JSP, MySQL, and Gemini API**.  
It allows users to ask questions, retrieves answers from a database if available, and if not, calls the **Google Gemini API** to generate a response. The new Q&A pair is then stored back into the database for future use.

### 🔑 Key Features
- **FAQ Database**: Stores common questions and answers in MySQL.  
- **Gemini Integration**: Automatically generates answers for new questions.  
- **Auto‑Learning**: Saves new Q&A pairs into the database.  
- **Responsive UI**: Built with JSP + Bootstrap for desktop and mobile.  
- **Clean Architecture**: Uses Spring Boot MVC, JPA/Hibernate, and service layers.  

### ⚙️ Tech Stack
- **Backend**: Spring Boot, JPA/Hibernate  
- **Frontend**: JSP, Bootstrap  
- **Database**: MySQL  
- **AI API**: Google Gemini
